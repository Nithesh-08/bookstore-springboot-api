📚 Bookstore Spring Boot API

This is a simple backend project built using Spring Boot to manage books and authors.  
I built this project to practice REST API development and understand how real backend systems are structured.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
🚀 Tech Stack
- Java  
- Spring Boot  
- Spring Data JPA  
- MySQL  
- Lombok  
- Maven  
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 📌 Features
- CRUD operations for Books and Authors  
- Pagination and sorting for books  
- Global exception handling  
- Clean layered architecture (Controller → Service → Repository)  
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 🧱 Project Structure
- Controller → Handles API requests  
- Service → Business logic  
- Repository → Database operations  
- Entity → Database models  
- DTO → Used for book data transfer  
- Exception → Handles errors globally  
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  API Endpoints
  📘 Books
- POST /api/books  
- GET /api/books  
- GET /api/books/{id}  
- PUT /api/books/{id}  
- DELETE /api/books/{id}  
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 ✍️ Authors
- POST /api/authors  
- GET /api/authors  
- GET /api/authors/{id}  
- PUT /api/authors/{id}  
- DELETE /api/authors/{id}
